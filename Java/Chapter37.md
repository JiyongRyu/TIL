| 구분                      | 배열 | Set 객체 |
| :------------------------ | :--: | :------: |
| 동일한 값을 중복하여 포함 |  ○   |    ✕     |
| 요소 순서의 의미          |  ○   |    ✕     |
| 인덱스로 요소에 접근      |  ○   |    ✕     |
